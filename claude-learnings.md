# Claude Learnings — Running Notes

This file is where reading becomes operational leverage. Every time you read something from the docs, a blog post, or Discord — if it would change how you work, write it down here with:

1. **Date**
2. **Source** (link or doc name)
3. **The insight** in one sentence
4. **The concrete change** you'll make — which workflow, skill, or CLAUDE.md gets updated, and how

**Rule:** Don't let this file become a highlight reel. If you can't name the concrete change, don't write it down — keep reading until you can.

**Monthly rhythm:** First Friday of the month, read through this file. Pick one entry you haven't acted on. Implement it.
